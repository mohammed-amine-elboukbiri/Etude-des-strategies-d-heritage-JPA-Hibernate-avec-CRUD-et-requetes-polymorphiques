#  Hibernate JPA Inheritances – Maven Project

##  Description

Ce projet montre comment utiliser **Hibernate + JPA** avec une base de données **H2** pour implémenter et tester les **trois stratégies d’héritage**.

Le projet inclut :

* ✔ Création d’un projet Maven
* ✔ Configuration Hibernate avec H2
* ✔ Implémentation des stratégies d’héritage JPA
* ✔ Tests CRUD
* ✔ Requêtes polymorphiques

---

##  Technologies utilisées

* **Java**
* **Maven**
* **Hibernate / JPA**
* **H2 Database**
* **JUnit (pour les tests)**

---

##  Stratégies d’héritage implémentées

###  Single Table

* Toutes les classes dans une seule table
* ✔ Rapide
* ❌ Beaucoup de colonnes null

###  Joined

* Tables séparées avec jointure
* ✔ Structure propre
* ❌ Requêtes plus lentes

###  Table per Class

* Une table par classe
* ✔ Indépendance
* ❌ Duplication de colonnes
